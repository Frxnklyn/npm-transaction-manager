# npm-transaction-manager

Koordiniert aufgeschobene Aenderungen verschiedener Module und uebernimmt oder verwirft sie gemeinsam ueber Commit und Rollback.

## Installation

```bash
npm install @frxnklyn/transaction-manager
```

## Exports

- `Updater` und `UpdaterInterface`
- `TransactionManager`
- `Transaction`
- `TransactionParticipantInterface`
- `TransactionState`
- `CommitError`
- `RollbackError`
